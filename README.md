# tms_aphasia

Naming task for patients with post-stroke aphasia.

## Task description:

- The session will last for 20 minutes, with each image appearing for a maximum of 20 seconds. If the patient names the image in less than 20 seconds, the operator moves on to the next word. If the patient fails to name the image, it disappears after 20 seconds and another one appears.
- There are 2 categories of images - Actions and Objects. The idea is to present them in blocks - a first block of objects and a second block of actions. The transition between blocks occurs at the 10-minute mark. The images are randomly selected within each block.
- The technician will verbally signal the start of each block and what it consists of (obviously after a pre-TMS introduction).
